Running Jobs
----

## See what jobs are running currently
```
jobs
```

## Kill a job
```
# Kill the 2nd job from job list
kill %2 
```

## Foreground job
```
# Bring 1st job back to foreground
fg 1
```

## Background job
Press `ctrl` + `z`


## Running command then keep promopt available
```
./abc.sh &
bg
```
